# Part 2: Current Process

## Opening Question Response
When a tenant reports a maintenance issue:

1. Tenant submits issue (email, phone, WhatsApp, or in person)
2. Admin logs request in Excel or property management software
3. Property manager reviews and assigns urgency
4. Assigned to maintenance staff or external vendor
5. Work is completed
6. Tenant notified
7. Cost recorded (sometimes delayed)
